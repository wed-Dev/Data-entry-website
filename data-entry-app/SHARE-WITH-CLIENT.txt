# SHARE APP WITH USA CLIENT - QUICK SETUP

## METHOD 1: Using Ngrok (Easiest - 5 Minutes)

### What is Ngrok?
Creates a secure tunnel so your USA client can access your local app through the internet.

### Setup Steps:

1. **Download Ngrok:**
   - Go to: https://ngrok.com/download
   - Sign up (free)
   - Download Windows version
   - Extract to: d:\Dubai\data-entry-app\

2. **Start Your Server:**
   Open PowerShell in d:\Dubai\data-entry-app\
   ```
   node server.js
   ```
   (Keep this running)

3. **Start Ngrok (in new PowerShell window):**
   ```
   cd d:\Dubai\data-entry-app
   ngrok http 3000
   ```

4. **Share URL with Client:**
   You'll see something like:
   ```
   Forwarding: https://abc123.ngrok-free.app -> http://localhost:3000
   ```
   
   Send this URL to your USA client:
   https://abc123.ngrok-free.app/app.html

5. **Client Access:**
   Your client opens that URL in their browser
   They can now use your system!

### Important Notes:
- ✅ Your computer must be ON
- ✅ Server must be running (node server.js)
- ✅ Ngrok must be running
- ✅ URL changes each time you restart ngrok (free version)
- ✅ Paid version ($8/month) gives permanent URL

## METHOD 2: Using Your Public IP (If You Have Static IP)

### Requirements:
- Static public IP address
- Router access
- Port forwarding knowledge

### Steps:

1. **Find Your Public IP:**
   - Go to: https://whatismyipaddress.com/
   - Note your IP (e.g., 123.45.67.89)

2. **Configure Router Port Forwarding:**
   - Login to your router (usually 192.168.1.1)
   - Forward port 3000 to your computer's local IP
   - (Google: "port forwarding [your router brand]")

3. **Update Server (for security):**
   Edit server.js to accept external connections

4. **Share with Client:**
   http://123.45.67.89:3000/app.html

### ⚠️ Security Warning:
This exposes your app to the internet. Add password protection!
